# 🚀 Borel Tec - Gestão de Produção Factory Flow (v7)


## 🛠️ Novidades da Versão 7.0

* **Perfis Independentes:** Cada usuário cadastrado possui seu próprio histórico, metas e cargas. Os dados de um perfil nunca se misturam com os de outro.
* **Sessão Permanente:** O login é mantido mesmo após fechar o navegador. O usuário só precisa deslogar manualmente.
* **Barra de Navegação (Nav):** Menu fixo para alternar entre funções de conta:
    * **Sair da Conta:** Encerra a sessão com confirmação de segurança.
    * **Deletar Conta:** Remove permanentemente o usuário e todos os seus dados do dispositivo.
* **Registro Simplificado:** Cadastro rápido e direto para facilitar o uso no ambiente de fábrica.

## 📋 Funcionalidades Consolidadas

* **Calendário Inteligente:** * 🟠 **Laranja:** Meta definida, produção em andamento.
    * 🟢 **Verde:** Meta atingida ou superada.
    * 🔍 **Consulta:** Clique em qualquer dia para ver o total produzido naquela data.
* **Controle de Cargas:** Funções de Pausa (⏸️), Edição (✏️) para conclusão parcial e Exclusão (🗑️).
* **Calculadora Integrada:** Ferramenta flutuante para contagem rápida de espumas.
* **Relatório WhatsApp:** Envio de resumo detalhado com horários formatados `(00:00 às 00:00)` e status de meta automático.

## 💾 Armazenamento e Segurança

Os dados são armazenados no `localStorage` do navegador utilizando chaves dinâmicas baseadas no nome de usuário (`bt_user_...`). 

> **Dica:** Para garantir que os dados estejam sempre salvos, finalize o relatório de produção ao término do turno.
